# SupportPilot AI

## Objective
Build an AI-assisted support ticket management system.

## Users
1. Customer
2. Support Agent

## Workflow

Customer submits ticket

↓

Backend stores ticket

↓

Gemini analyzes ticket

↓

Gemini returns

- Category
- Priority
- Sentiment
- Suggested Reply

↓

Dashboard displays ticket

↓

Agent edits reply

↓

Agent marks ticket resolved
